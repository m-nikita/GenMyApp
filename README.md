# GenMyApp

## Serveur

### Installation
npm install express mongoose openai cors dotenv jsonwebtoken bcrypt
npm install --save-dev nodemon

### Exécution
npm run dev
OU
nodemon server.js

## Client

### Installation
#### Angular
npm install --global @angular/cli

#### Boostrap
npm install --save boostrap

### ngx-clipboard
npm install ngx-clipboard

### Exécution
npm run start
OU
ng serve
